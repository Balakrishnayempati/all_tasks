HTML(Hypertext markup language)

1.Hypertext ---------- Collection of text and links.


2.MARKUP ---------- Gives instructions to the browser to dispaly the data.


3.LANGUAGE ---------- Interaction.

HTML was created by Tim bernes lee and organized by W3C(world web consortium).

STRUCTURE OF HTML

DOCTYPE
Root element
head
body

Tags

The keyword which is enclosed between less than and greater than symbol to display the data in browser and every tag has its own behaviour.

Types of tags

1.Paired tags (or) Container tags

The tag which is having opening and closing tags.

Example:p,h1-h6,div,span...

2.Unpaired tags (or) Empty tags

The tag which is closing itself is known as self-closing tags .

Example:img,br,hr,input,iframe...

Basic Structure of html

!DOCTYPE html -- Defines latest version of html

html -- Parent of all tags

head -- Gives information about document

title -- Gives the title to our browser

body -- Display the given content on the browser

Browser in html -- Browsers never accept spaces or line breaks between the content in HTML.

Elements

An element is the basic building block used to Structure and dispaly content on the web.

1.Opening Tag

2.Content

3.Closing Tag

Elements can have attributes also

Types of elements

1.BLOCK-LEVEL ELEMENTS

The element which occupies whole width and creates a new line

Example:h1-h6,p,div,semantic elements

2.INLINE-BLOCK ELEMENTS

The element which occupies only required space and does not create a new line in css we can apply width,height,padding top and bottom and margin top and bottom.

Example:img,input,button,iframe

3.INLINE ELEMENTS

The element which occupies only required space and does not create a new line in css we cannot apply width,height,padding top and bottom and margin top and bottom.

Example:span,formatting tag,b,strong,italic
